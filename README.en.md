# Game Lab – Build Games, Learn Web Development

🇩🇪 [Deutsche Version](README.md)

A growing collection of small browser games built with [Phaser 3](https://phaser.io/). This project is part of a programming course designed to teach young learners web development and game design.

## About

Each game introduces new concepts and builds on previous lessons. By recreating well-known arcade classics step by step, students learn:

- **HTML, CSS & JavaScript** — the foundations of web development
- **Game loops** — how `update()` drives frame-by-frame logic
- **Player input** — keyboard controls for movement and actions
- **Physics & collision detection** — using Phaser's Arcade Physics
- **Game state management** — tracking score, game over conditions, and restart flow
- **Sprite and texture management** — loading and displaying images and animations
- **Enemy AI & game logic** — movement patterns, randomness, and difficulty scaling

## Tech Stack

- **Phaser 3** (loaded via CDN)
- Vanilla HTML, CSS & JavaScript
- No build tools or bundler required – just open in a browser

## Getting Started

1. Clone the repo
2. Open any `.html` file in a browser (or use a local server like Live Server)
3. Play!
